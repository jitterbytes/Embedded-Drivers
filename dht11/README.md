# DHT11 Driver

Platform-independent driver for the DHT11 temperature and humidity sensor.

## 📌 Overview
This driver is designed to be portable across multiple embedded platforms by separating core logic from hardware-specific implementation.

## ✨ Features
- Platform-independent core driver
- Simple and clean API
- Designed for portability across frameworks like Arduino and ESP-IDF

## 📂 Structure

- `driver/` → Core driver implementation  
- `platform/` → Platform-specific GPIO and timing implementations  
- `examples/` → Example usage for different platforms

Initial structure created. Implementation in progress.
